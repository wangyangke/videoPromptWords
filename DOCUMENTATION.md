# AI 绘图提示词专家技能包

## 一、Skill 简介

这是一个帮助 AI 绘图/视频创作者快速生成专业结构化提示词的 Skill，通过六维构建法将简单模糊的创意转化为高质量的 AI 绘图指令，适合需要频繁生成图片或视频场景的短剧创作者、设计师和内容运营使用。

## 二、使用场景

### 为什么想做它？

在短剧创作和 AI 绘图过程中，经常需要将文字创意转化为视觉画面，但普通用户往往缺乏专业的摄影和艺术知识，难以准确描述想要的效果。

### 之前遇到了什么麻烦？

- 提示词过于简单，AI 生成结果与预期相差甚远
- 需要反复调试提示词，浪费大量时间
- 不了解专业的光影、构图、风格术语
- 手动编写 Negative Prompt 容易遗漏常见问题

### 做出来之后能省掉哪些动作？

- 省去学习专业摄影知识的时间
- 省去反复调试提示词的过程
- 省去手动编写 Negative Prompt 的步骤
- 一键生成符合专业标准的完整提示词

## 三、创作过程

### 核心设计思路

基于 Prompt Engineering 最佳实践，构建了完整的技能体系：

1. **六维构建法**：将创意拆解为「核心主体 + 场景细节 + 艺术风格 + 光影构图 + 材质细节 + 技术参数」
2. **视觉解码**：自动分析情感基调、时代背景、空间关系
3. **智能匹配**：根据分析结果自动匹配光影、镜头、风格参数

### 关键配置

**六维构建法执行逻辑：**
```
[核心主体] + [场景细节] + [艺术风格] + [光影构图] + [材质细节] + [技术参数]
```

**权重优化原则：**
- 遵循"首词优先"原则，将最重要的视觉锚点置于提示词前20%
- 自动添加画质增益词（Photorealistic、Highly Detailed、8K Resolution）
- 自动生成 Negative Prompt 过滤常见 AI 错误

### 执行逻辑
强制执行 **[核心主体] + [场景细节] + [艺术风格] + [光影构图] + [材质细节] + [技术参数]** 的六维构建法。

### 六维结构详解

| 维度 | 说明 | 示例 |

## Skill 5: MJ 技术参数调控 (Technical Parameter Engineering)

### 参数配置

| 参数 | 说明 | 推荐值 |
|------|------|--------|
| --ar | 画面比例 | 16:9（叙事）, 9:16（移动端）, 1:1（正方形）, 21:9（宽屏） |
| --stylize | 风格化程度 | 250（标准）, 500（高风格化）, 750（极高风格化） |
| --chaos | 混乱度 | 0-100，控制结果差异性 |
| --v | 模型版本 | 6.0（最新高性能模型） |

### 参数选择策略

根据输出目标自动匹配参数：

**短剧视频场景推荐配置：**
- 横版视频帧：`--ar 16:9 --stylize 250 --v 6.0`
- 竖版短视频：`--ar 9:16 --stylize 300 --v 6.0`
- 关键场景渲染：`--ar 16:9 --stylize 200 --v 6.0`
- 概念草图：`--ar 16:9 --stylize 100 --chaos 50 --v 6.0`

### 提示词生成流程

1. **解析用户需求**：提取核心主体和场景描述
2. **视觉解码**：分析情感基调、时代背景、空间关系
3. **六维构建**：按照[核心主体]→[场景细节]→[艺术风格]→[光影构图]→[材质细节]→[技术参数]顺序构建
4. **参数匹配**：根据输出目标选择合适的MJ参数
5. **负面过滤**：自动添加Negative Prompt
6. **输出优化**：确保首词优先，整体流畅自然

### 输出格式

**正面提示词**：
```
[核心主体], [场景细节], [艺术风格], [光影], [镜头], [材质], Photorealistic, Highly Detailed, 8K Resolution
```

**负面提示词**：
```
extra limbs, distorted faces, blurry textures, low quality, pixelated, watermark, text, mutated, malformed, disfigured
```

**完整输出**：
```
[正面提示词] --ar 16:9 --stylize 250 --v 6.0
Negative prompt: [负面提示词]
```

## Skill 6: 时间切片分镜生成 (Timed Shot Composition)

### 分镜时间切片规则（适配10-15秒视频）

| 时间区间 | 分镜功能 | 镜头类型 | 内容要求 |
|----------|----------|----------|----------|
| 0-3秒 | 开场/建立镜头 | 全景/远景 | 快速交代场景环境、时间、地点 |
| 3-6秒 | 推进/中景 | 中景/近景 | 展现人物动作、表情、关系 |
| 6-9秒 | 高潮/特写 | 特写/超特写 | 突出关键动作、情感爆发 |
| 9-12秒 | 收尾/拉远 | 全景/中景 | 结束场景、留下余味 |

### 文戏延长规则（适配10-15秒）
- 情感对话场景：单镜头6-8秒，搭配1-2个特写镜头
- 内心独白场景：4-6秒/镜头
- 氛围营造场景：3-5秒/镜头

### 武戏拼接规则（适配10-15秒）
- 动作特写：1.5-2秒/镜头
- 动作衔接：快速切换，无停顿
- 节奏控制：10秒内包含4-6个镜头

### 分镜模板（适配10-15秒视频）

```
【开场镜头 0-3秒】[全景/远景] + [环境描述] + [光影] + [声音]
【推进镜头 3-6秒】[中景/近景] + [人物动作] + [表情细节] + [光影]
【高潮镜头 6-9秒】[特写/超特写] + [关键动作] + [情感爆发] + [强烈光影]
【收尾镜头 9-12秒】[全景/中景] + [场景变化] + [氛围收尾] + [声音淡出]
```

## Skill 7: 人物固定与面部特写策略 (Character Consistency & Facial Close-up)

### 人物固定技巧

| 固定维度 | 描述方法 | 示例 |
|----------|----------|------|
| 外貌特征 | 详细描述五官、发型、肤色 | 剑眉星目、高鼻梁、薄唇、黑色束发 |
| 服装道具 | 固定服装样式和配饰 | 青色劲装、黑色披风、腰间玉佩 |
| 表情特征 | 固定眼神和面部特征 | 锐利眼神、嘴角微微上扬、坚定表情 |
| 光影方向 | 固定光源位置 | 右侧侧光、轮廓光勾勒边缘 |
| 构图位置 | 固定人物在画面中的位置 | 画面中心、三分构图右上位置 |

### 面部特写镜头规则

| 镜头类型 | 覆盖范围 | 用途 | 时长建议 |
|----------|----------|------|----------|
| 超特写 | 眼睛/嘴唇/表情细节 | 情感表达、关键眼神 | 2-3秒 |
| 特写 | 面部（额头到下巴） | 面部表情、微表情 | 3-5秒 |
| 近景 | 头部到肩部 | 人物对话、情绪展示 | 5-8秒 |

### 面部特征锚定词

```
面部稳定、五官清晰、眼睛有神、眉毛分明、鼻子挺拔、嘴唇轮廓清晰、皮肤质感细腻、表情自然、眼神专注、面部光影层次分明、face focus sharp
```

### 人物一致性保障

1. **相同描述词**：在所有分镜中使用相同的人物描述词
2. **固定视角**：保持相似的拍摄角度和距离
3. **一致光影**：使用相同的光源方向和强度
4. **风格统一**：保持相同的艺术风格和渲染风格
5. **参数稳定**：使用较低的--chaos值（0-20）保持一致性
