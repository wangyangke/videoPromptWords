# AI 绘图提示词专家 Skill 介绍文档

---

## 1、Skill 简介

这是一个帮助 AI 绘图/视频创作者快速生成专业结构化提示词的 Skill，通过六维构建法将简单模糊的创意转化为高质量的 AI 绘图指令。专用于短剧视频场景生成，支持时间切片分镜（0-5秒/5-10秒）、细节动作描述、四类参考素材（形象/环境/动作/声音），文戏延长武戏拼接，确保面部稳定不变形、画面无卡顿。

---

## 2、使用场景

### 为什么想做它？
在使用 MidJourney、Stable Diffusion 等 AI 绘图工具时，经常遇到以下问题：
- 不知道如何描述才能让 AI 生成符合预期的图像
- 提示词过于简单导致生成结果质量低下
- 反复调整提示词浪费大量时间
- 不了解专业的光影、构图、风格术语
- 生成的人脸变形、画面卡顿

### 之前遇到的麻烦？
- 输出结果与预期相差甚远
- 需要反复尝试才能找到合适的提示词组合
- 缺乏专业的摄影和艺术知识
- Negative Prompt 需要手动添加，容易遗漏常见问题
- 生成的视频帧面部不稳定、有卡顿现象

### 做出来之后能省掉哪些动作？
- 省去学习专业摄影知识的时间
- 省去反复调试提示词的过程
- 省去手动编写 Negative Prompt 的步骤
- 省去逐帧调整面部和画面质量的工作
- 一键生成符合专业标准的完整分镜提示词

## 3、创作过程

### 核心设计思路

基于 Prompt Engineering 最佳实践，构建了十二维技能体系：

| 技能模块 | 核心功能 |
|----------|----------|
| 结构化 Prompt 建模 | 六维构建法：主体+场景+风格+光影+材质+参数 |
| 视觉摄影与光影控制 | 专业灯光库（12种光效）和镜头库（12种镜头） |
| 艺术媒介与风格合成 | 多风格迁移（16种风格）和艺术家对标（12位艺术家） |
| 画质精修与负面工程 | 自动增益修正和逆向过滤（30+负面词） |
| MJ 技术参数调控 | 智能参数匹配（比例、风格化、混乱度） |
| 场景类型与情感匹配 | 12种场景类型适配矩阵 + 细节动作转换 |
| 时代背景与地域文化适配 | 6个时代 + 6个地域文化特征 |
| 角色与人物细节库 | 年龄状态 + 情绪表达（细节动作版） |
| 环境与场景元素库 | 自然环境 + 城市环境（含声音参考） |
| 天气与时间特效 | 8种天气 + 7种时间特效（含声音参考） |
| 时间切片分镜生成 | 0-5秒/5-10秒/10-15秒/15-20秒分镜规则 |
| 四类参考素材库 | 形象、环境、动作、声音素材库 |

### 关键配置

**核心优化规则：**
1. **时间切片分镜**：按0-5秒/5-10秒等时间节点生成分镜
2. **细节动作代替抽象情绪**：用具体动作描述替代抽象情绪词
3. **四类参考素材**：精准使用形象、环境、动作、声音素材
4. **文戏延长武戏拼接**：文戏场景延长停留时间，武戏场景快速拼接
5. **质量锚定语**：每段提示词必加"面部稳定不变形、画面无卡顿"

**六维构建法执行逻辑：**
```
[核心主体] + [场景细节] + [艺术风格] + [光影构图] + [材质细节] + [技术参数]
```

**分镜时间切片规则：**
| 时间区间 | 分镜功能 | 镜头类型 | 内容要求 |
|----------|----------|----------|----------|
| 0-5秒 | 开场/建立镜头 | 全景/远景 | 交代场景环境、时间、地点 |
| 5-10秒 | 推进/中景 | 中景/特写 | 展现人物动作、表情、关系 |
| 10-15秒 | 高潮/特写 | 特写/超特写 | 突出关键动作、情感爆发 |
| 15-20秒 | 收尾/拉远 | 全景/远景 | 结束场景、留下余味 |

### 情感到动作转换示例

| 抽象情绪 | 细节动作描述 |
|----------|--------------|
| 悲伤 | 指尖轻轻划过眼角，肩膀微微颤抖，低头注视地面 |
| 喜悦 | 嘴角上扬露出牙齿，眼睛弯成月牙，身体轻轻跳跃 |
| 愤怒 | 眉头紧锁成川字，拳头紧握指节发白，呼吸急促 |
| 恐惧 | 瞳孔收缩放大，身体向后蜷缩，双手护住胸口 |
| 浪漫 | 指尖轻轻触碰，目光温柔对视，身体微微靠近 |

## 4、使用步骤

### 调用方式

1. **触发方式**：在 SOLO 中直接描述您的创意场景
2. **格式要求**：自然语言描述即可，无需特定格式

### 操作步骤

```
步骤 1：打开 SOLO，进入技能中心
步骤 2：启用「AI 绘图提示词专家」Skill
步骤 3：在聊天框中输入您的场景描述，例如：
        "一个小女孩在花园里跳舞"
步骤 4：Skill 自动解析并生成专业分镜提示词
步骤 5：复制生成的完整指令到 AI 绘图工具中使用
```

```

```

- 过于简单，缺乏细节描述
- AI 无法准确理解场景
- 生成结果可能不符合预期
- 没有面部稳定和画面质量保障


## 7、总结与思考

### 收获与效率提升

通过使用这个 Skill，创作者可以获得以下提升：

1. **效率提升**：提示词生成时间从平均 5-10 分钟缩短到几秒钟
2. **质量提升**：生成图片的精准度和艺术质量显著提高
3. **知识门槛降低**：无需专业摄影和艺术知识也能生成专业级提示词
4. **一致性保障**：每次生成的提示词都遵循统一的专业标准
5. **面部稳定保障**：确保生成的人脸不变形、不扭曲
6. **画面流畅保障**：避免画面卡顿、闪烁等问题
7. **分镜生成**：自动按时间切片生成分镜，便于视频创作

### 最满意的地方

1. **时间切片分镜**：按0-5秒/5-10秒等时间节点自动生成分镜
2. **细节动作转换**：将抽象情绪词转换为具体动作描述
3. **四类参考素材**：形象、环境、动作、声音素材库
4. **文戏武戏处理**：文戏延长、武戏快速拼接
5. **质量锚定语**：确保面部稳定不变形、画面无卡顿
6. **负面过滤增强**：新增人脸变形、画面卡顿等负面词

### 后续优化方向

1. **多语言支持**：增加中英文双语提示词生成
2. **风格模板库**：预设更多场景风格模板
3. **动态参数调整**：根据用户反馈智能调整参数
4. **视频分镜生成**：支持生成完整的视频分镜脚本
5. **用户自定义词库**：允许用户添加自定义的风格和词汇
6. **实时预览功能**：根据提示词生成缩略图预览
