# AI 绘图提示词专家 Skill

一个帮助 AI 绘图/视频创作者快速生成专业结构化提示词的工具，通过六维构建法将简单模糊的创意转化为高质量的 AI 绘图指令。

## ✨ 核心功能

- **结构化 Prompt 建模**：六维构建法（主体+场景+风格+光影+材质+参数）
- **视觉摄影与光影控制**：专业灯光库和镜头库
- **艺术媒介与风格合成**：多风格迁移和艺术家对标
- **画质精修与负面工程**：自动增益修正和逆向过滤
- **MJ 技术参数调控**：智能参数匹配
- **时间切片分镜生成**：适配 10-15 秒视频平台限制
- **人物固定与面部特写**：保持人物一致性

## 🎯 适用人群

- 短剧创作者
- 视频内容运营
- UI/UX 设计师
- AI 绘图爱好者

## 🚀 快速开始

```bash
# 1. 克隆项目
git clone https://github.com/wangyangke/videoPromptWords.git

# 2. 打开 SKILL.md 查看完整技能配置

```

## 📝 使用示例

**输入：**
```
森林小屋中的神秘老人
```

**输出：**
```
【分镜1 0-3秒】全景：幽深的森林中，一座古朴的木屋在晨雾中若隐若现，木质栅栏上爬满藤蔓，屋顶炊烟袅袅升起...

【分镜2 3-6秒】中景：一位身穿灰色长袍的神秘老人坐在木屋门前的摇椅上，银白长发披肩、深邃眼眸...

【分镜3 6-9秒】特写：老人抬起头望向镜头，眼神深邃神秘、嘴角微微上扬...

【分镜4 9-12秒】收尾：老人低下头继续翻阅书籍，水晶球放在旁边的石桌上发出柔和的光芒...

Negative prompt: extra limbs, distorted faces, blurry textures...
```

## 📋 分镜时间切片规则

| 时间区间 | 分镜功能 | 镜头类型 |
|----------|----------|----------|
| 0-3秒 | 开场/建立镜头 | 全景/远景 |
| 3-6秒 | 推进/中景 | 中景/近景 |
| 6-9秒 | 高潮/特写 | 特写/超特写 |
| 9-12秒 | 收尾/拉远 | 全景/中景 |
