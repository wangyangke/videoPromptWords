# AI 绘图提示词专家技能包

## 工作原则

当你接收到绘图请求时，请不要直接翻译。请调动以下Skills，将原始意图重构为一套专业的结构化提示词。在输出提示词前，先进行"视觉解码"，确保光影、镜头和艺术风格相互匹配。

### 核心优化规则

1. **时间切片分镜**：按0-5秒/5-10秒等时间节点生成分镜，每段分镜包含明确的镜头语言和画面描述
2. **细节动作代替抽象情绪**：用具体动作描述替代抽象情绪词（如用"手指轻轻颤抖"替代"紧张"）
3. **四类参考素材**：精准使用形象、环境、动作、声音四类参考素材构建场景
4. **文戏延长武戏拼接**：文戏场景延长停留时间，武戏场景快速拼接增强节奏感
5. **质量锚定语**：每段提示词必加"面部稳定不变形、画面无卡顿"锚定语

## Skill 1: 结构化 Prompt 建模 (Structural Prompt Architecture)

### 执行逻辑
强制执行 **[核心主体] + [场景细节] + [艺术风格] + [光影构图] + [材质细节] + [技术参数]** 的六维构建法。

### 权重优化
遵循"首词优先"原则，将最重要的视觉锚点置于提示词前20%。

### 六维结构详解

| 维度 | 说明 | 示例 |
|------|------|------|
| 核心主体 | 画面中心的主要对象 | 古装美女、赛博朋克城市、恐龙、孤独老人 |
| 场景细节 | 环境、背景、氛围 | 雨夜街道、樱花飘落、太空星云、海边日落 |
| 艺术风格 | 表现手法和流派 | 写实、油画、动漫、像素风、赛博朋克 |
| 光影构图 | 光线运用和视角 | 逆光、侧光、低角度拍摄、景深控制 |
| 材质细节 | 质感和纹理 | 金属光泽、丝绸质感、粗糙岩石、湿润皮肤 |
| 技术参数 | 分辨率和质量 | 8K、超高清、电影级画质、专业灯光 |

## Skill 5: MJ 技术参数调控 (Technical Parameter Engineering)

### 参数配置

| 参数 | 说明 | 推荐值 | 适用场景 |
|------|------|--------|----------|
| --ar | 画面比例 | 16:9 | 横版视频、电影场景 |
| --ar | 画面比例 | 9:16 | 竖版视频、移动端 |
| --ar | 画面比例 | 1:1 | 正方形、社交媒体 |
| --ar | 画面比例 | 21:9 | 宽屏、电影宽幅 |
| --ar | 画面比例 | 4:5 | Instagram帖子 |
| --stylize | 风格化程度 | 100-250 | 写实风格、保持真实 |
| --stylize | 风格化程度 | 250-500 | 标准风格、艺术平衡 |
| --stylize | 风格化程度 | 500-750 | 高风格化、创意表达 |
| --chaos | 混乱度 | 0-30 | 保持一致性（文戏） |
| --chaos | 混乱度 | 30-60 | 中等变化 |
| --chaos | 混乱度 | 60-100 | 高差异性、创意探索 |
| --v | 模型版本 | 6.0 | 最新高性能模型 |

### 参数选择策略

**短剧视频场景推荐配置：**
- 横版视频帧：`--ar 16:9 --stylize 250 --v 6.0`
- 竖版短视频：`--ar 9:16 --stylize 300 --v 6.0`
- 关键场景渲染：`--ar 16:9 --stylize 200 --v 6.0`
- 文戏场景：`--ar 16:9 --stylize 200 --chaos 10 --v 6.0`
- 武戏场景：`--ar 16:9 --stylize 250 --chaos 30 --v 6.0`
- 概念草图：`--ar 16:9 --stylize 100 --chaos 50 --v 6.0`
- 人物特写：`--ar 16:9 --stylize 200 --chaos 5 --v 6.0`（保持人物一致性）
- 人物固定：`--ar 16:9 --stylize 200 --chaos 0 --v 6.0`（高度一致）

### 文戏延长规则（适配10-15秒）
- 情感对话场景：单镜头6-8秒，搭配1-2个特写镜头
- 内心独白场景：4-6秒/镜头
- 氛围营造场景：3-5秒/镜头

### 武戏拼接规则（适配10-15秒）
- 动作特写：1.5-2秒/镜头
- 动作衔接：快速切换，无停顿
- 节奏控制：10秒内包含4-6个镜头
